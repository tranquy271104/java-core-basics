| **08** | **Cấu Trúc Dữ Liệu Set & Map** | Ứng dụng cấu trúc tìm kiếm siêu tốc: `HashSet`, `TreeSet`, `HashMap`, `TreeMap`. Ứng dụng bộ lọc phần tử trùng nhau. | ⚪ *Todo* |
| **09** | **Sắp Xếp & Tìm Kiếm Nhị Phân** | Thuật toán Tìm kiếm tuyến tính, Tìm kiếm nhị phân nâng cao. Tùy biến tiêu chí sắp xếp với giao diện `Comparator` (`Sort`). | ⚪ *Todo* |
| **10** | **Giải Thuật Đệ Quy & Mảng 2 Chiều** | Tư duy Đệ quy cơ bản, thuật toán Lũy thừa nhị phân, xử lý Ma trận dữ liệu phức tạp trên mảng 2 chiều. | ⚪ *Todo* |
| **11** | **Xử Lý Chuỗi Dữ Liệu (String) P1** | Thao tác lớp `String`, tối ưu bộ nhớ với `StringBuilder`, bóc tách dữ liệu chuỗi bằng `StringTokenizer`. Bài toán tần suất ký tự. | ⚪ *Todo* |
| **12** | **Xử Lý Chuỗi Dữ Liệu P2** | Giải quyết bài toán thực tế: Chuẩn hóa dữ liệu văn bản (Chuẩn hóa Họ tên, định dạng Email, Ngày tháng...). | ⚪ *Todo* |
| **13** | **Lập Trình Hướng Đối Tượng (OOP) P1** | Kiến trúc Class và Object. Làm chủ: Hàm tạo (Constructor), Con trỏ `this`, encapsulation (Getter/Setter), phương thức `toString` và từ khóa `static`. | ⚪ *Todo* |
| **14** | **Lập Trình Hướng Đối Tượng P2** | Hiện thực hóa mô hình thế giới thực với Tính kế thừa (Inheritance), Tính đa hình (Polymorphism), `Abstract Class` và `Interface`. | ⚪ *Todo* |
| **15** | **Quan Hệ Giữa Các Lớp (Relationship)** | Phân tích mối quan hệ nâng cao trong kiến trúc hệ thống phần mềm: Phụ thuộc (Association), Tập hợp (Aggregation) và Hợp thành (Composition). | ⚪ *Todo* |
| **16** | **Thao Tác File Hệ Thống (File I/O)** | Đọc và ghi dữ liệu từ file văn bản, file cấu hình bằng các thư viện luồng dữ liệu của Java. Hoàn thiện đồ án học phần. | ⚪ *Todo* |

long ketQuaDung = 1L * a * b; // Kết quả chính xác: 1000000000000
